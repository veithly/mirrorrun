# MirrorRun Combined Pitch-Demo Composition

This folder is the HyperFrames-ready visual composition workspace for the Devpost single-video delivery mode.

Current status:

- Deck source exists at `pitch/deck.html`.
- Deck thumbs are generated by `npm run deck:render`.
- Live app screenshots are generated by `npm run screenshots`.
- A repeatable raw capture path exists: `PORT=3100 npm run record:demo`.
- Current raw capture artifacts exist at `pitch/recording/raw.webm`, `pitch/recording/cues.json`, and `pitch/recording/raw-preview.png`.
- Pitch narration is generated with StepFun through `npm run narration:pitch`; the mp3 files live in `pitch/polish-combined/assets/`.
- Composition QA is `npm run combined:check` plus `npm run combined:contact-sheet`.
- Final render is blocked until sponsor keys are installed, because the HackathonHunter realness gate requires live provider evidence rather than a video made entirely from credential blockers.

Planned final outputs:

- `pitch/recording/pitch-demo-combined.mp4` — silent HyperFrames visual render.
- `pitch/recording/pitch-demo-combined-final.mp4` — final ffmpeg audio mux with StepFun narration and BGM.
- YouTube unlisted URL copied into `SUBMISSION.md`.

Do not add audio tags to the HyperFrames HTML. Narration and music are muxed exactly once after the silent visual render.
